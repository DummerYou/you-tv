# 更新日志

## v1.0.0 - 2026-06-25

`you-tv` 是一款面向 Android TV 和手机横屏使用的 IPTV 播放应用，支持 TXT、M3U、JSON 频道源导入，内置频道分组、收藏、多源切换、EPG 节目单、代理、开机启动和远程配置页面。

本版本是首个公开自动发布版本，适合用于首次安装和基础体验。

### 主要功能

- 支持 Android 10+ 设备安装使用。
- 支持 IPTV 频道源导入和地址源自动更新。
- 支持中文频道源文本的 UTF-8、GB18030/GBK 编码兼容。
- 支持 IPv4、IPv6、HTTPS、域名视频地址。
- 支持同频道多播放源，播放失败后可自动尝试下一源。
- 支持遥控器、键盘和手机触控操作。
- 支持 EPG 节目单和远程配置网页。

### 下载

请在本 Release 下方的 Assets 区域下载 `you-tv.apk`。

`Source code (zip)` 和 `Source code (tar.gz)` 是 GitHub 自动生成的源码压缩包，不是安装包。

### 安装说明

下载 `you-tv.apk` 后，将 APK 复制到 Android TV 或 Android 手机上安装。首次安装时，系统可能需要允许“安装未知来源应用”。
